EXTERNAL ACCOUNT CREATION LINKS
────────────────────────────────────────────────────────────────────────────────

Supabase (Required)
  Sign up: https://supabase.com
  After creating project:
    • Settings → API → Copy Project URL as SUPABASE_URL
    • Settings → API → Copy "anon" key as SUPABASE_ANON_KEY
    • Settings → API → Copy "service_role" as SUPABASE_SERVICE_ROLE_KEY

Lichess.org (Optional - for integration)
  Token: https://lichess.org/account/oauth/token
  Create Personal Access Token for LICHESS_API_TOKEN

Redis Cloud (Optional - alternative to local)
  Upstash: https://upstash.com
  Provides REDIS_URL and REDIS_PASSWORD
